# Web-Term-Project
## Description
This website is developed to raise awareness about homelessness and highlight the challenges faced by homeless individuals. The platform provides information about causes of homelessness, its impact on society, and possible ways people and organizations can help.

The website is designed with a clean and simple layout to ensure easy navigation and accessibility for all users.

## Features
- Informative content about homelessness
- Causes and effects of homelessness
- Ways to help and support homeless communities
- Clean, professional, and user-friendly design
- Responsive layout for different screen sizes

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Purpose
This project aims to educate users about homelessness and encourage social responsibility by spreading awareness through a digital platform.
